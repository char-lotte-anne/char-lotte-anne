## Repo structure (for technical readers)

| Path | Purpose |
|------|---------|
| `index.html` | Single-page layout |
| `styles.css` | Theming (DM Sans, Fraunces), dark mode, accessibility (contrast; focus outline only on keyboard via `:focus-visible`) |
| `script.js` | Nav menu (toggle, focus trap, Escape), scroll reveal, back-to-top, contact (email/phone copy), section hash, theme toggle |
| `vercel.json` | Deploy config: favicon build step, security headers, resume download behavior |
| `favicon.svg` | Source for favicons; script generates PNGs and `icon.ico` from it |
| `icon.ico` / `icon.png` / `favicon-16x16.png` / `apple-touch-icon.png` | Generated by `npm run favicon` from `favicon.svg` (Vercel runs this on deploy). All referenced in HTML. |
| `safari-pinned-tab.svg` | Static mask icon for Safari pinned tab (different format; not generated). Referenced in HTML. |
| `resume.pdf` | Resume (downloadable from site) |
| `images/` | Hero portrait, experience photos, project previews (all referenced in `index.html`) |
| `scripts/` | Favicon generator: `icon.png`, `favicon-16x16.png`, `apple-touch-icon.png`, `icon.ico` from `favicon.svg` (sharp, svg-to-ico) |
